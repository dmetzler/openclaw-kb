# Agent-Reach - CLI web access pour agents IA

## Résumé

Outil open-source (scaffolding) qui donne aux agents IA un accès unifié à Internet : Twitter, Reddit, YouTube, GitHub, Bilibili, XiaoHongShu — un CLI, zéro frais d'API.

## Points clés
- Compatible Claude Code, OpenClaw, Cursor, Windsurf
- Architecture modulaire : chaque plateforme = un "channel" indépendant
- 17+ plateformes supportées
- Principalement développé pour le marché chinois mais couvre aussi les plateformes occidentales
- Installation en une commande pour l'agent
